## vim ping.yml

```
---
- hosts: all
  tasks:
  - name: ping
    ping:
  - name: echo
    command: echo "hello ansible"
```


## ansible & playbook

```
ansible -i HOST, all -u USER -m ping
ansible -i HOST, all -u USER -a "echo hello,ansible"

ansible-playbook -i HOST, -u USER ping.yml
```
